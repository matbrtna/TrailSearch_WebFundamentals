# TrailSearch_WebFundamentals

## Overview

This was my first programing semestral project in my first semester at uni. The goal of this project was to create PHP app base on chosen topic. All of the pages are in czech language same as documentation that was mandatory.

## Project topic

For the topic I have chosen to create web app with map of bike trails, where each participant could review the trails that were posted.

## Users

1. Admin - Can create new trail pages and delete comments or other trail pages.
2. Register user - Can review the trails.
3. Unregistered user - Can searh the trails and only read the reviews.
